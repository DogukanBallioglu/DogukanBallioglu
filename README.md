### 🚀 Neler Yapıyorum?

*   **Full-Stack Geliştirme:** C# ASP.NET Core ve React kullanarak kurumsal ödeme, hakediş ve yönetim sistemleri mimarileri kuruyorum.
*   **Mobil Geliştirme:** React Native ile kişisel finans ve bütçe takip uygulamaları tasarlıyorum.
